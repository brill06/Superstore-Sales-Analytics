# Superstore Sales Performance Analysis

## Project Overview
This project analyzes sales data from a global superstore to identify high performing product categories, regional trends, and potential areas of concern for management.

## Dataset
- Source: Kaggle - Superstore Sales Datasets
- File: superstoresales.csv
- Records: 10,000 orders

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Google Colab

## Key Questions Answered
- Which product categories generate the most revenue?
- How sales have changed over time
- How is the Technology category performing in the West region?

## Key Insights
- Technology is the top revenue contributor
- Sales show clear year over year trends
- Technology sales in the West region show variability and require further investigation
